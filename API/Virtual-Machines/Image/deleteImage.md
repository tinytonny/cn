# deleteImage


## 描述
删除一个私有镜像，只允许操作您的个人私有镜像。<br>
若镜像已共享给其他用户，需先取消共享才可删除。


## 请求方式
DELETE

## 请求地址
https://vm.jdcloud-api.com/v1/regions/{regionId}/images/{imageId}

|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**imageId**|String|True||镜像ID|
|**regionId**|String|True||地域ID|

## 请求参数
无

|名称|类型|描述|
|---|---|---|
