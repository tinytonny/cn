# 产品功能

## 海量数据计算

提供企业级数据仓库解决方案，提供SQL/PySpark等计算能力，满足海量数据ETL以及交互式查询需要。

## 完全托管开箱即用

简单设置即可快速开展数据分析工作，集群底层搭建及运维完全托管。

## 任务开发
 
提供任务开发，自定义函数，任务调度功能，支持开发人员通过WEB/IDE/CLI/SDK进行任务开发。

## 用户权限管理

对数据存储和访问制定安全机制，对数据操作权限具备多维权限校验机制
