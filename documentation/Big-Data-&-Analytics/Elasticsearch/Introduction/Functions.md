# 产品功能

## 兼容开源生态产品
支持Kibana、Logstash等产品对接
开放的API，开放大量RestfulAPI，与生态产品完美对接，预置插件，主流分词、索引插件植入

## 集群高可用
## 支持异地灾备
提供异地灾备并保证服务可用性，默认提供多分区、多副本，保证数据索引效率和稳定性，保证后台的稳定监控以及持续集群监控和告警能力

## 数据安全保障
支持用户拥有独立集群
用户专有网络，专有网络内访问，杜绝外网直连风险，保证数据隐私
