# 部署作业

您可以通过登录 JMR 控制台，点击“作业管理”菜单。
本文介绍如何部署作业。

## 操作步骤
1.登录 [JMR 控制台](https://xdata.jdcloud.com/rmgr/resources/res-manage/custom-resources.html#/)，选择查看哪个地域下的集群。

2.创建
 - 选择执行策略：单次执行和周期执行。
 - 选择调度时间。
 - 勾选要部署的作业。
 
3.相关操作说明
 - 立即执行
    - 点击立即执行部署的作业。
 - 修改
    - 对作业的部署进行相似或重新配置。
 - 暂停
    - 暂停所部署的作业。
 - 删除
    - 直接删除所部署的作业。
 - 任务详情
    - 点击任务名称，可以跳到任务详情页面同时可以查看执行历史情况。
 - 执行历史
    - 点击任务名称，可以跳到任务详情页面同时可以查看执行历史情况。
