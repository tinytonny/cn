# 删除库
根据您的业务变化，您可以手动删除数据库实例的库。 

## 前置条件
* 只有实例状态为 **运行** 的时候，才能进行删除操作。
* 开启同步服务时候创建的库不允许手动删除。

## 操作步骤
1.  登录 [区块链数据服务控制台](https://bds-console.jdcloud.com/block/list)。 
2.  选择目标实例所在地域。 
3.  点击目标实例，进入实例详情页，切换至 **库管理**  Tab 页，找到要删除的数据库，点击操作列 **删除** 按钮。
4.  在确认弹窗中，点击 **确定** 删除库。
