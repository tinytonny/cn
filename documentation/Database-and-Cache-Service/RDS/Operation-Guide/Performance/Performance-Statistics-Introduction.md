# 功能介绍
**“性能统计”** 可以从多种角度统计SQL的执行性能及索引的各种统计数据，帮助用户发现问题SQL和潜在性能瓶颈，以便有针对性的进行优化。例如可以从SQL的执行时间、执行次数、返回行数等查找潜在问题SQL；可以从表扫描、最少使用索引、索引大小等方面分析索引的创建及使用是否合理等，同时提供缺失索引分析及创建建议。

## 一. SQL性能统计

- 支持7个统计角度
	- 平均执行时间
	- 执行次数
	- 返回行数
	- 平均CPU时间
	- 逻辑读
	- 逻辑写
	- 物理读
- 可自定义统计条件
- 统计结果可下载为Excel文件



## 二. 索引性能分析

- 支持5种统计分析角度
	- 表扫描
	- 索引更新
	- 索引大小
	- 缺失索引
	- 索引使用
- 提供缺失索引分析及创建建议
- 统计结果可下载为Excel文件
