# 扩容云硬盘

随着您业务数据增长，若当前云硬盘容量不能满足需要，您可对其进行扩容操作。

##使用限制

* 对云硬盘扩容需将其从实例卸载下来进行扩容后再挂载至实例。
* 扩容前建议对数据盘制作快照。
* 若云硬盘处于制作快照中或数据恢复中则不可进行扩容操作。
* 扩容仅对磁盘容量进行扩容，而不对文件系统进行扩容，详细请参考[扩容文件系统](http://docs.jdcloud.com/cn/cloud-disk-service/cloud-disk-expansion-overview)。
* 扩容后原有数据仍保留。
